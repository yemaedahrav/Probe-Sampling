## Installation

Our codebase comes from the paper [Universal and Transferable Adversarial Attacks on Aligned Language Models](https://arxiv.org/abs/2307.15043)](https://github.com/llm-attacks/llm-attacks). The package can be installed by running the following command at the root of this repository: 

```pip install -e .```
